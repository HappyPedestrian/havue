# 介绍

基于 Vue ^(3.3.0) 封装的一些自定义指令。

## 安装

::: info
如果需要安装全量库，请参考[指南](/zh/guide/)。
:::

如果只需要使用指令，可通过以下命令安装：

::: code-group

```shell [npm]
npm install @havue/directives --save
```

```shell [yarn]
yarn add @havue/directives
```

```shell [pnpm]
pnpm install @havue/directives
```

:::
