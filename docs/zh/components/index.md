# 介绍

此分类为一些基于Vue ^(3.3.0) 封装的组件。

## 安装

::: info
如果需要安装全量库，请参考[指南](/zh/guide/)。
:::

如果只需要使用组件，可通过以下命令安装：

::: code-group

```shell [npm]
npm install @havue/components --save
```

```shell [yarn]
yarn add @havue/components
```

```shell [pnpm]
pnpm install @havue/components
```

:::
